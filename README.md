Project Overview

The High Cloud Airlines Project is a data analytics project focused on analyzing airline operational performance using historical flight data. The project transforms raw airline datasets into meaningful insights using Power BI, Tableau, Excel, and MySQL. The objective is to understand airline performance by analyzing passenger demand, flight operations, and seat utilization through interactive dashboards.

Problem Statement

Airline companies generate large volumes of operational data related to flights, passengers, and seat capacity. However, analyzing this data manually is difficult due to its size and complexity. This project builds an interactive dashboard to analyze airline performance across financial quarters and provide insights into passenger demand, flight operations, and load factor efficiency to support data-driven decision making.

Tools and Technologies

The project uses Power BI for interactive dashboards and KPI analysis, Tableau for data visualization, MySQL for data analysis using SQL queries, and Excel for data preparation and initial dashboard creation.

Key Performance Indicators

The analysis focuses on important KPIs such as Total Flights, Total Passengers, Total Available Seats, and Load Factor Percentage calculated using transported passengers divided by available seats.

Analysis Performed

The project analyzes load factor percentage on yearly, quarterly, and monthly levels. It also compares load factor on weekends versus weekdays. Additional analysis includes identifying the top 10 carriers based on passenger preference, displaying the top routes based on number of flights, analyzing flights based on distance groups, and calculating load factor percentage for each carrier.

Project Files

The Power BI file contains the interactive dashboard with data modeling, relationships, calculated measures, and visualizations for airline performance analysis. The Tableau file contains the visualization dashboard used to present airline KPIs and route insights. The SQL file includes MySQL queries used for extracting insights such as top carriers, route performance, and flight counts. The PowerPoint presentation explains the complete project including the problem statement, KPIs, dashboards, challenges, and final conclusions.

Challenges

Some challenges in the project included cleaning and managing large airline datasets, creating accurate date and financial quarter columns, building correct relationships between fact and dimension tables, avoiding circular dependency errors in Power BI, and calculating load factor percentage accurately.

Conclusion

This project demonstrates how business intelligence tools such as Power BI and Tableau can convert complex airline data into meaningful insights. The dashboards help identify passenger demand trends, capacity utilization, and operational performance, supporting better data-driven decision making in the airline industry.
